# 项目概述 python344

本项目是一个基于Python开发的Web应用，结合了Flask框架和MySQL数据库技术，实现了一套图书借阅管理系统。

# 技术栈

- 开发语言：Python
- Web框架：Flask
- 数据库系统：MySQL
- 开发工具：PyCharm
- 数据库连接工具：Navicat

# 功能模块

## 用户功能

- 图书借阅
- 图书归还

## 管理员功能

- 图书管理

## 可视化功能

- 数据展示

# 系统角色

- 用户
- 管理员

# 运行环境

- 操作系统：Windows
- 数据库系统：MySQL

# 部署步骤

1. 使用PyCharm导入项目
2. 安装依赖包
3. 配置MySQL数据库
4. 运行项目

# 目录结构

```
(project root)
│
├── app             # Flask应用目录
│   ├── models.py   # 数据模型
│   ├── views.py    # 视图处理
│   ├── templates   # HTML模板
│   └── static      # 静态文件
│
├── venv            # 虚拟环境目录
│
└── requirements.txt  # 依赖包列表
```

# 核心亮点

- 分角色设计，满足不同用户需求
- 基于MySQL的稳定数据存储
- 简便的部署流程
